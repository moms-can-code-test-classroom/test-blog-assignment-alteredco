In case we haven’t met, hi lovely people, I'm Wendy. I'm Student Success Manager at Moms Can Code School and a frontend web developer that loves good design, JavaScript and Python! I recently taught Intro to Python with Moms Can & Co and my current endeavours include taking on the JS Full-Stack Tech degree and occasionally opening a French textbook. I love working in education and believe that code can foster boundless creativity, active engagement, inclusivity and open communication.
